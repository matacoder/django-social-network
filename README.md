## Technical Stack

- Python
- Django
- Pytest

## Description

This project is made during my education at Yandex Data School (aka Yandex.Practicum).

## Features

- Registration and authentication,
- Posts,
- Comments,
- Image uploads,
- Forms and model based forms,
- Pagination,
- Flatpages,
- Followings,
- Posts' groups,
- Dynamic navigation,
- Covered with tests (TDD).
